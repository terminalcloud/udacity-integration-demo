Embedding Terminals in Udacity Classroom

  1. Drag this button: Embed Terminal    to your Chrome Bookmarks Bar.

    You can try this out on this udacity project page.

    If you need to do so, click View...Always Show Bookmarks Bar to show the bookmarks bar.
    Watch this video for a walkthrough of this step:
    Embed Project Embed Terminal Embed Editor
  2. Log in to Udacity using your @udacity.com email address.
  3. Visit a Udacity classroom page, for example, this one.
  4. Click the bookmarklet.
    You will see an embedded Terminal in the Udacity classroom. This is the same embedded Terminal that Udacity CD's can embed with Coco content.

Inserting Terminal Javascript Snippets in Coco

  1. Drag this button: Insert Snippets     to your Chrome bookmarks bar.
    If you need to do so, click View...Always Show Bookmarks Bar to show the bookmarks bar. (Watch this video for a walkthrough of this step.)
  2. Visit Coco and click inside a content textarea.
  3. Click the Insert Snippet bookmarklet.
